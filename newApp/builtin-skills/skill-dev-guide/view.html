📖 Skill 开发指南

最简结构

my-skill/
└── SKILL.md  ← 唯一必需文件

带可视面板

my-skill/
├── SKILL.md      ← 技能定义
└── view.html     ← 右侧面板页面

SKILL.md 格式

---
name: 我的技能
description: 一句话描述
---

技能详细说明写在这里。
AI 调用时会读取这段内容。

消息通信

宿主 → view.html

→ skill-data 首次加载，携带完整数据
→ skill-update 增量更新数据
→ skill-progress 进度通知 (current / total)

view.html → 宿主

← skill-callback 自定义回调 { action, data }

代码示例

// 接收数据
window.addEventListener("message", (e) => {
  if (e.data?.type === "skill-data") {
    const data = e.data.payload;
    // 渲染你的页面...
  }
});

// 发送回调
window.parent.postMessage({
  type: "skill-callback",
  action: "my-action",
  data: { key: "value" }
}, "*");
